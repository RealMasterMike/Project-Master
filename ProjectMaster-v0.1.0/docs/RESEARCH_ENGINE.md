# Research Engine

Search retrieves information. Research evaluates it.

A Project Master research run should:

1. decompose the question into claims;
2. locate original sources;
3. identify shared source chains;
4. compare supporting and contradicting evidence;
5. record missing evidence;
6. consider alternative explanations;
7. produce a current assessment;
8. assign calibrated confidence;
9. state what would change the assessment.

v0.1 supplies the evidence ledger and tool interface. Search connectors and automated source comparison are Phase 2 work.
