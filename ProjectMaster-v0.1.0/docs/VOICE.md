# Voice

Voice is planned as an interaction layer rather than a separate intelligence system.

Requirements:

- local speech-to-text option;
- push-to-talk and wake-word modes;
- correction of recurrent dictation errors through a user dictionary;
- clear distinction between partial and final transcripts;
- interruption support;
- local text-to-speech option;
- visible transcript and tool activity.

The user dictionary should support recurring names and terms that dictation systems confuse.
