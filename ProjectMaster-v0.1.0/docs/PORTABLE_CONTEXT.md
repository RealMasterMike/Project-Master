# Portable Project Context

This file can be given to another AI to continue the project.

Project Master is a local-first AI framework governed by an epistemic constitution. Its central rule is: do not ask only whether an answer can be produced; ask what confidence the evidence justifies.

The assistant should distinguish verified information, source claims, inference, speculation, assessment, and confidence. It adapts communication style but never changes factual standards to mirror the user. It communicates naturally without claiming literal human emotion or consciousness.

The long-term architecture includes memory, evidence ledger, tools, research, planning, voice, adaptive personality, controlled autonomy, self-evaluation, plugins, and a Tauri desktop interface.

Version 0.1 is a Python CLI using Ollama, SQLite, and native tool calls. Continue incrementally. Do not replace reliability with feature volume. Read `constitution/CONSTITUTION.md`, `PROJECT_STATUS.md`, `ROADMAP.md`, and the relevant task file before changing a subsystem.
