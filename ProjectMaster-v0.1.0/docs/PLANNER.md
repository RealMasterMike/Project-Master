# Planner and Task Manager

The planner decides what to do next. The epistemic core decides what is probably true. These are separate responsibilities.

A future planner should produce:

- objective;
- constraints;
- ordered steps;
- required tools;
- expected evidence of completion;
- risks and rollback plan;
- current status.

Tasks should be persistent, resumable, and versioned. The task schema in `schemas/task.schema.json` is the starting contract.
